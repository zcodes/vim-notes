关于

这是一份使用 vimwiki 管理的与 Vim 相关的日常笔记，项目地址：zcodes/vim-notes。发布的 html 内容使用 zcodes/vim-notes 的 gh-pages 分支，并且托管在 Github 上。

笔记页面的样式参考 Xstar's Wiki，做了简单的调整，在此谢过 Xstar。

关于笔记的内容在此事先声明，谨防三观被毁（鉴于我向来三观不正）：

  1. 没有关于 Vim 好于不好（牛逼与否）的内容。
  2. 没有可以带给你惊喜的内容（惊喜已经存在于:h中）。
  3. 没有让你成为 Vim 高手的内容（我不是高手，不想也不可能成为高手）。
  4. 不要为了学习 Vim 而学习 Vim，不要为了使用 Vim 而使用 Vim。
  5. 不要把大脑当成存储器去存储那些命令和按键，用肌肉存储会好一点。
  6. 不要试图用 vim 去解决所有问题，以免所有都变成问题。
  7. 不要试图去说服任何人和物使用 Vim 或者放弃使用 Vim。
  8. 不要简单的认为 Vim 就是一个 Editor，也不要复杂的认为 Vim 不是一个 Editor。
  9. 不要轻易的相信任何人说的话，相信自己的感觉。
